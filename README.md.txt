# Projeto de Business Intelligence (BI)

## Descrição

Este projeto foi desenvolvido como parte da disciplina de **Business Intelligence (BI)** do curso de [Seu Curso] na [Sua Instituição].

O objetivo é demonstrar um fluxo básico de coleta e visualização de dados, utilizando **Python**. O projeto é dividido em duas etapas principais:

1. **Geração de dados simulados** em formato `.csv`.
2. **Construção de um dashboard interativo** com base nesses dados.

---

## Estrutura do Projeto

- `gerar_dados.py`: Script responsável por gerar um arquivo `.csv` com dados simulados.
- `dashboard.py`: Script que lê os dados do `.csv` e apresenta visualizações interativas utilizando bibliotecas como `pandas`, `matplotlib`, ou `plotly` (dependendo do que foi usado).
- `requirements.txt`: Lista de dependências necessárias para executar o projeto.
- `README.md`: Este arquivo.
